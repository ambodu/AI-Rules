# AI-Rules

AI 驱动的产品开发方法论与规则体系。

## 核心命题

1. **反馈机制** — 什么样的设计能保证反馈本身正确，最终产品正确？
2. **产品设计** — 如何设计产品来保证正确性，最终达成更好用户体验？
